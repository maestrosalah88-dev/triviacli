# Trivia CLI Quiz

This is a command-line interface (CLI) trivia quiz built using JavaScript and Node.js.

The quiz presents the user with multiple-choice questions and tracks the score and time taken.

## Features

* Timed gameplay (60 seconds)
* Scoring logic
* End-game percentage calculation and feedback messages (e.g., "EXCELLENT!", "GREAT JOB!")

## Setup and Running the Quiz

1.  *Prerequisites:* Ensure you have *Node.js* installed on your system.
2.  *Clone the repository:* bash
    git clone [https://github.com/maestrosalah88-dev/triviacli.git](https://github.com/maestrosalah88-dev/triviacli.git)
    cd triviacli
    
3.  *Run the program:*
    bash
    node index.js
